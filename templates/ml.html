Machine Learning

Supervised Machine Learning

On a labelled data set, supervised machine learning trains itself. That is, the data is labelled with information that the machine learning model is designed to identify, and it may even be classed in the manner that the model is designed to categorised data. A computer vision model trained on a data collection of varied tagged dog photos, for example, may be used to detect purebred German Shepherd dogs.

Because the model's outcomes can be compared to real labelled outcomes, supervised machine learning uses less training data than other machine learning approaches and makes training easier. However, correctly labelled data is costly to prepare, and there's a risk of overfitting, or constructing a model that's so dependent on and biased by the training data that it can't handle fresh data changes appropriately.

Unsupervised Machine Learning

Unsupervised machine learning takes large amounts of unlabelled data and employs algorithms to extract relevant characteristics that may be used to label, sort, and categorise the data in real time, without the need for human interaction. Unsupervised learning is more about discovering patterns and correlations in data that people might overlook than it is about automating judgments and predictions.

Consider spam detection: humans create far more email than a team of data scientists could ever identify or categorise in their lives. An unsupervised learning system can sift through large amounts of data to find traits and patterns that signal spam (and keep getting better at flagging spam over time).
