Experiments

Experiment 1

Dataset : Secret Recipe

Description :

This experiment is conducted with the dataset contains around 3000 data. The data are all text data and they're all about the review on the restaurant named Secret Recipe. The dataset used here had gone through the data cleaning process to ensure the dataset is clean to be train. Once the dataset is cleaned, the emotion extraction function will occur to get the emotions from the dataset. The emotions would be features of the training process, meanwhile the GMM method would be used to get the label of the experiment from the emotion. After that, both features and labels would be train to fit into different models.

Experiment 2 and 2.1

Dataset : Secret Recipe

Description :

This experiment was conducted with the same dataset in experiment 1. In experiment 2, the whole process will be the same as experiment 1 but there is an extra step. This extra step is about removing the restaurant name in the dataset since the restaurant name will affect the accuracy of the models. In experiment 2.1, the texts those do not provide emotion value would be remove to make sure the model can provide more accurate results.

Experiment 3

Dataset : Secret Recipe

Description :

This experiment did what experiment 2.1 did. But the data in the dataset would be translated into English language. Since the dataset contains texts from multiple languages or even mixed languages in a sentence, the data had to be translated into a specific language to ensure the the emotion extraction function did not missed any important data that could affect the value of the emotions.
