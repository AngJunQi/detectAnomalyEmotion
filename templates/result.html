Metrics

Accuracy: It is calculated by dividing the number of correct predictions by the total number of predictions.

Precision: It is defined as the proportion of correct predictions to the total number of correct predictions.

Recall: It is the ratio of positive predictions to the number of positive class value in the test data.

F1 Score: It represents the balance of precision and recall.

Confusion Matrix

Confusion matrix is super useful for measuring the accuracy, precision, recall and more. Inside the boxes, it contains TP, FP, FN and TN. The meaning of these words will be present in below:

• True Positive (TP) – Predicted result is true and actual result is true.

• False Positive (FP) – Predicted result is true and actual result is false.

• True Negative (TN) – Predicted result is false and actual result is true.

• False Negative (FN) – Predicted result is false and actual result is false.
